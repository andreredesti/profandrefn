Linguagem de Marcação - HTML

O HTML é uma linguagem de marcação (e não uma linguagem de programação, pois não possui lógica de execução) usada para criar e estruturar o conteúdo de uma página web. Ele funciona como o esqueleto de um website, definindo a organização dos elementos na tela.

  • Marcação: O HTML utiliza tags (como <p> para parágrafo ou <img> para imagem) para "marcar" e escrever o conteúdo, informando ao navegador o que cada parte representa (título, lista, link, etc.).
  • Hipertexto: O "HT" de HTML refere-se à capacidade de conectar documentos através de hiperlinks, o que é essencial para a navegação e a própria interligação de toda a web.

É bom lembrar que com o HTML não há a preocupação com cores, ações e nem interatividade da página. Para deixamos a página ou aplicação "elegante" faremos uso do CSS e para criarmos interatividade usaremos o Javascript, PHP, Python ou outra linguagem de programação.

O HTML pode ser escrito a partir de um bloco de notas ou usando-se um editor de texto. Com o HTML podemos inserir os dados para uma página WEB ou outra aplicação que o utilize para estruturação da mesma. O propósito do HTML é o de estruturar os textos, parágrafos, imagens e demais elementos da aplicação ou página WEB.

  • Estrutura do HTML

    O HTML funciona como uma caixa dentro de outra caixa. Uma caixa maior e dentro dela caixas menores. A caixa principal engloba as demais caixas que forem criadas dentro dela.

    Podemos criar arrumações diversas dentro da caixa principal do HTML. Para isso precisamos pensar em qual será o Layout da nossa aplicação.

  • Principais TAGS

    Algumas das principais tags do HTML

    • A tag <html> é a mãe de todas tags de uma página. Ela é inserida no começo e no final do documento web. Dentro desta tag ficará todo o conteúdo da aplicação. A tag <html> possui abertura e fechamento: <html></html>

    • Na estruturação do documento html há várias tags para se conhecer. A tag <head> é responsável pelas informações de comunicação no navegadores de internet. Ela possui informações que não é vista pelo usuário mas são importantes para transferência de dados entre as várias aplicações WEB. São inseridas dentro dela os metadados. Várias outras tags ficarão dentro dela.

    • O conteúdo principal da aplicação ficarão dentro da tag <body></body>. O cabeçalho, o conteúdo principal, as diversas seções, os títulos e texto, as imagens e demais elementos do documento ficarão entre essa tag.

    • Para títulos e subtítulos podemos usar a tag <h>. Ela possue um número que pode ser de 1 a 6. Para o título principal usa-se <h1> e para os subtítulos os demais números. Ela também tem abertura e fechamento.

    • A tag <p> é utilizada na criação de parágrafos. Um texto que for inserido na página pode usar essa tag. Ela também possui abertura e fechamento. A tag <p> possui abertura e fechamento: </p>
